# Weather App

Ứng dụng thời tiết đơn giản được xây dựng bằng React, sử dụng OpenWeatherMap API.

## Tính năng

- Xem thời tiết của các thành phố tại Việt Nam
- Hiển thị thông tin thời tiết bằng tiếng Việt
- Danh sách các thành phố phổ biến để truy cập nhanh

## Cài đặt

1. Clone repository:

```bash
git clone https://github.com/namh33868/Weather-API.git
```

2. Cài đặt dependencies:

```bash
npm install
```

3. Tạo file .env trong thư mục gốc và thêm API key của bạn:

```
REACT_APP_WEATHER_API_KEY=your_api_key_here
```

4. Chạy ứng dụng:

```bash
npm start
```

## Công nghệ sử dụng

- React
- Axios
- OpenWeatherMap API
